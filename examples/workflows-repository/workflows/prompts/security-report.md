# Security Report Generation

Generate a security findings report based on the analysis.

## Report Structure

### Executive Summary
- Overall security posture assessment
- Critical findings count

### Findings

For each finding:
1. **Severity**: Critical / High / Medium / Low
2. **Description**: What is the issue?
3. **Location**: Where in the code?
4. **Recommendation**: How to fix it?

### Recommendations
- Prioritized list of remediation steps

## Constraints

- **You have exactly 1 turn** - generate the complete report
- No tools available - just generate the report text
- This is the final output that will be posted to Linear
